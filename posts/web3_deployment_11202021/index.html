Deploy solidity contract with web3

November 20, 2021

Example of deploying a solidity contract with web3 on a test Ganache blockchain.

Quick example that follows https://github.com/smartcontractkit/full-blockchain-solidity-course-py#lesson-4-web3py-simple-storage

I previously created a simple contract that mimics a store inventory. I now need to deploy the contract with the web3 python package.

Compile solidity contract

We first need to compile the solidity contract. Below function reads the solidity program, compiles, and returns the ABI and compiled bytecode. We need the ABI and bytecode to deploy a new instance of the contract.

main.py

from web3 import Web3
from solcx import compile_standard, install_solc
import os
from dotenv import load_dotenv
import json

def compile_solidity_contract(_contract_fp: str, _contract_name: str):

    # read contents of the solidity contract file
    with open(_contract_fp, "r") as f:
        contract_file = f.read()

    # compile the contract code
    compiled_sol = compile_standard(
        {
            "language": "Solidity",
            "sources": {_contract_fp: {"content": contract_file}},
            "settings": {
                "outputSelection": {
                    "*": {
                        "*": [
                            "abi",
                            "metadata",
                            "evm.bytecode",
                            "evm.bytecode.sourceMap",
                        ]
                    }
                }
            },
        },
        solc_version="0.6.0",
    )

    with open("compiled_code.json", "w") as file:
        json.dump(compiled_sol, file)

    # the abi and bytcode are important to create contracts
    bytecode = compiled_sol["contracts"][_contract_fp][_contract_name]["evm"][
        "bytecode"
    ]["object"]

    abi = json.loads(
        compiled_sol["contracts"][_contract_fp][_contract_name]["metadata"]
    )["output"]["abi"]

    # save abi to disk
    with open("abi.json", "w") as f:
        json.dump(abi, f)

    return (abi, bytecode)

Create test blockchain

We need a running blockchain to deploy our contract. Ganache is a simple tool to locally run a test blockchain.

Ganache gives us different test accounts with 100 ether. Choose one account and create a .env file to store the credentials.

Deploy contract

The contract is deployed as a transaction to the ledger. It is interesting that we need to include a nonce when we build the transaction. Nonce is equal to the number of transactions an account has made.

main.py

def deploy_contract(abi, bytecode):

    chain_id = int(os.getenv("CHAIN_ID"))
    my_address = os.getenv("ADDRESS")
    private_key = os.getenv("PRIVATE_KEY")
    
    # get connection to ganache
    w3 = Web3(Web3.HTTPProvider(os.getenv("RPC_URL")))
    
    # Create the contract
    contract = w3.eth.contract(abi=abi, bytecode=bytecode)
    
    # Contract is deployed as a transaction to the ledger. 
   
    # 1) Build transaction 
    # nonce = number of transactions an address has made. 
    nonce = w3.eth.getTransactionCount(my_address)
    transaction = contract.constructor().buildTransaction(
        {"chainId": chain_id, "from": my_address, "nonce": nonce}
    )
    # 2) Sign transaction w/ private key
    signed_txn = w3.eth.account.sign_transaction(transaction, private_key=private_key)
    
    # 3) Send transaction
    tx_hash = w3.eth.send_raw_transaction(signed_txn.rawTransaction)
    tx_receipt = w3.eth.wait_for_transaction_receipt(tx_hash)
    return tx_receipt.contractAddress

Call the above functions in a main method:

if __name__ == "__main__":

    # assumes 0.6.0 compiler is already installed
    # install_solc("0.6.0")

    load_dotenv()

    contract_fp = "store.sol"
    contract_name = "Store"

    abi, bytecode = compile_solidity_contract(contract_fp, contract_name)
    contract_address = deploy_contract(abi, bytecode)
    print(contract_address)

We can now interact with the deployed contract! As a recap, the contract represents a store inventory. We can add new products to the store. Each product has a name and price.

Get contract reference

interact.py

First get a reference to the deployed contract. We need the contract address and ABI.

def get_contract(address, abi_fp):

    w3 = Web3(Web3.HTTPProvider(os.getenv("RPC_URL")))

    with open(abi_fp, "r") as f:
        abi = json.load(f)
    contract = w3.eth.contract(address=address, abi=abi)

    return contract

Interact with contract

The deployed contract has a addProduct function that accepts a product name and price and adds it to an internal array of products. Below function builds a transaction that executes the contract function with user input.

# add a new product to our inventory
def add_product(contract, product: str, price: int):

    w3 = Web3(Web3.HTTPProvider(os.getenv("RPC_URL")))
    sender_address = os.getenv("ADDRESS")
	
    # build transaction that calls contract's addProduct function
    transaction = contract.functions.addProduct(product, price).buildTransaction(
        {
            "chainId": int(os.getenv("CHAIN_ID")),
            "from": sender_address,
            "nonce": w3.eth.getTransactionCount(sender_address),
        }
    )
    # sign transaction w/ private key
    signed_transaction = w3.eth.account.sign_transaction(
        transaction, private_key=os.getenv("PRIVATE_KEY")
    )
    # send transaction
    transaction_hash = w3.eth.send_raw_transaction(signed_transaction.rawTransaction)
    tx_receipt = w3.eth.wait_for_transaction_receipt(transaction_hash)
    return tx_receipt

Call the above functions in a main method:

We first get a reference to the deployed store contract. Next we add table and gpu to the store’s inventory (represented as a public array called products). Finally we view the store’s products.

if __name__ == "__main__":

    load_dotenv()

    contract_address = "0x1c5CCf422C982798D1367F3E0480d7C7fb0E1c09"
    abi_fp = "abi.json"

    contract = get_contract(contract_address, abi_fp)
    transaction_receipt = add_product(contract, "table", 200)
    transaction_receipt = add_product(contract, "gpu", 20000)

    # confirm
    print(contract.functions.products(0).call())
    print(contract.functions.products(1).call())
